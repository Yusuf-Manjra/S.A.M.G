## F.A.Q

Why Are There Two Blueprints Folders?
> The main blueprint folder contains any blueprint that are also tied to C++ code. \
> The other blueprint folder, "Pure Blueprints" is specifically for objects that have no C++ code tied to them, these are mainly prototype objects.
